# Scenarios

Short pieces about agents acting on your behalf in worlds that haven't quite arrived yet.

These are scenarios, not predictions. Each one starts with a vivid case — a missing delivery, a midnight trade, a courier handoff on a rooftop — and ends with the small thing that nobody noticed was missing from the agent economy until it was too late to add quietly. The architecture beneath each piece is shipped and runnable today; the story is a way of pointing at what that architecture is *for*.

If you want the dry version, read [The Agent Identity Papers](../articles/). If you want the dream version, read these.
